HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Class and id tags are used to identify certain elements within you html.
  • Semantic elements such as
    help specify exactly what type of element it is.
  • At the start of the html document, it is required to place to specify what code is being written.
  • The Title tags are used in header and is only visible on the top of the browser tab.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • background-color changes the color of the background
  • Width and height is used in css to change the size of an element
  • box-shadow is used to create a shadow around a boxed element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add: Moves changes from the working directory to the staging area
  • git clone: Creates a copy of an existing Git repository
  • git commit: Takes the staged snapshot and commits it to the project history, usually combined with git add
  • git config: A convenient way to set configuration options for your git installation
  • git merge: A powerful way to integrate changes from divergent branches
  • git pull: It downloads a branch from a remote repository and immediately marges it into the current branch
  • git push: Lets you move local branch to another repository
  • git status: This command displays the state of the working directory and the staged snapshot
